# 📱 Social Media Dashboard

A modern and responsive Social Media Dashboard developed using HTML, CSS, JavaScript, and Chart.js. This dashboard displays social media analytics, follower statistics, engagement reports, and interactive charts.

---

## 🚀 Project Overview

The Social Media Dashboard provides a centralized interface to monitor social media statistics from different platforms such as Instagram, Twitter, YouTube, and Facebook. The dashboard includes interactive cards, live statistics, charts, notifications, and responsive design.

---

## ✨ Features

- Dark Theme Dashboard
- Responsive Design
- Interactive Social Media Cards
- Real-Time Digital Clock
- Chart.js Analytics Graph
- Popup Information Windows
- Statistics Update Button
- Random Followers Update
- Activity Notifications
- Sidebar Navigation
- Dark/Light Mode Toggle
